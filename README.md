```sh
yarn dev
```

## .env file

This should be the structure of the .env file:

```
GITHUB_ID=
GITHUB_SECRET=
GOOGLE_ID=
GOOGLE_SECRET=
NEXT_AUTH=https://example.com
JWT_SECRET=
```

You can generate a good value to <b>JWT_SECRET</b> on the terminal via this openssl command.

```
openssl rand -base64 32
```
